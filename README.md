# Muay Thai Landing Page

A responsive landing page project built to practice front-end web development fundamentals and UI design.

## Live Demo
🔗 https://stevieblu.github.io/muaythai/

## Overview
This project is a simple landing page inspired by Muay Thai fitness/training brands.

The goal was to strengthen my skills in:
- Semantic HTML structure
- CSS styling and layout
- Responsive design
- Basic JavaScript interactions
- Git/GitHub deployment with GitHub Pages

## Features
- Hero section
- Call-to-action buttons
- Responsive layout for desktop/mobile
- Clean modern fitness aesthetic

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Purpose
This is an early portfolio project created as part of my web development learning journey.

I plan to continue improving this project by adding:
- Better animations/interactions
- Pricing or membership sections
- Contact form
- Backend/API integration
- Full multi-page site conversion

## Installation
Clone the repo and open locally:

```bash
git clone https://github.com/yourusername/muaythai.git
cd muaythai
```

Then open `index.html` in your browser.
